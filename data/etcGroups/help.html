Help:
Select Single Click on an item to expand the network of its connections.
Double Click on an item to open the related website.
Drag Click and drag an item to reposition it.
Pan Left-click the background and drag it to pan the graph.
Zoom Right-click and drag the mouse up or down to zoom the graph.
Zoom-To-Fit Right-click once to zoom the display to fit the whole graph.
Search Search type :
    Global (default) : will select any group containing the search string.
    Exact : will select the groups that match exactly the search string.
    Prefix : will select the groups that start with the search string.
Search on :
    Select the fields you want to search on.
Search :
    Enter a search string and type <return>.
